# SIGAC - Sistema de Gestão Acadêmica

> [!WARNING]
> Este projeto está em fases preliminares de desenvolvimento! No momento não está utilizável.

## Sobre
O SIGAC é uma aplicação web que gerencia dados de discentes de uma instituição de ensino permitindo cadastro, atualização e consulta de registros dos estudantes.
## Tecnologias
Implementado em Javascript Vanilla utilizando as bibliotecas _ExpressJS_ e _better-sqlite3_ em conjunto com o sistema de banco de dados SQLite3.
## Como começar
```bash
$ git clone https://github.com/kprincipe/sigac-web
$ cd sigac-web/
$ npm start
```
